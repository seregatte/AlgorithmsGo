# AlgorithmsGo

Some Algorithms written in Go for language studies (PR are welcome)

## Requirements

- Go

## Algorithms

| Name 						          | Solutions 								            | Description 										                  |
|---------------------------|---------------------------------------|---------------------------------------------------|
| ReverseString			        | 1 								                    | Reverse a given string            								|       

Dependencies
------------

None


Run tests
----------------

Inside a algorithms directory run that command:

```shell
$ go test
```

or 

```shell
$ make
```
